Create amazing solutions with just a single prompt. Join the challenge and show your creative side!

Purpose

The One-Prompt Challenge Workshop aims to analyze the power of AI to create different kind of solutions in one prompt. The goal of the challenge is to create a solution to a problem or idea using a single prompt.

Challenge How-To

  1. Clearly identify a problem or idea you wish to explore.
  2. Select a suitable starter template.
  3. Define your target user clearly.
  4. Generate your first version using the chosen prompt.
  5. Review your outcome, iterate, and improve your prompt within the provided attempts.
  6. Submit your prompt on this web page

# Generic Interactive Game Prompt Template

Create an interactive, engaging web game app designed for users to [briefly describe target audience or age group]. The game should clearly introduce a playful scenario or character related to [state your chosen theme or topic here]. Follow this structured approach:

1. Introduction & Visual Appeal

 * Briefly introduce the main scenario or character.

 * Provide clear, intuitive instructions.

2. Gameplay & Challenges

 * Include interactive, intuitive, progressively challenging tasks.

 * Immediate feedback or points awarded for correct actions.

3. Conclusion & Rewards

 * Celebrate user achievements clearly and engagingly.

 * Offer simple rewards (badges, scores, animations).

### Technical: Colorful, visually engaging, responsive design.

# Generic Productivity App Prompt Template

Develop a simple and interactive web-based productivity app tailored for [briefly state target audience, e.g., professionals, students]. The app should effectively help users with [clearly state productivity goal or challenge]. Structure the app clearly:

1. Clear Introduction

    * Briefly outline purpose and benefits.

    * Interactive Functionality

2. Simple, intuitive interactions guiding users towards their goals.

    * Visual or interactive elements for motivation and progress tracking.

3. Completion & Encouragement

    * Offer encouraging feedback upon task completion.

    * Clearly display progress summaries or motivational visuals.

### Technical: Clean, intuitive UI, fully responsive for desktop and mobile.

# Generic Learning Game Prompt Template

Create a gamified educational web app for [target audience ] to learn the [topic] interactively. Follow the structure:

Fun Examples:

- Friendly cartoon character introduces concept with real life examples.

- Present 1-2 clear examples visually showing correct triangle solutions step-by-step.

1. Visual Theory Explanation

    * Briefly and visually explain the underlying theory with real-world examples.

2. Interactive Challenges

    * Offer 2-3 tasks where learner apply the theory.

    * Immediate humorous feedback; award badges for correct answers.

3. Quiz

    * Short (1-2 questions) quiz to reinforce learning with fun feedback.

### Technical: Bright visuals, humorous and engaging interactions, responsive for all devices.
